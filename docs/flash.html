♟ OpenChess Web Installer

Flash your ESP32 directly from the browser — no IDE needed

1 Requirements

  • A Chromium-based browser (Chrome, Edge, Opera) — Firefox and Safari are not supported
  • An ESP32 dev board connected via USB
  • USB drivers installed for your board (CP2102 or CH340)

2 What gets flashed

  • Bootloader — ESP32 boot code
  • Partition table — memory layout with OTA support
  • Firmware — OpenChess application
  • Web UI — chess board interface (LittleFS filesystem)
⚠️ First-time flash This will erase any existing data on the ESP32. After flashing, the board will start in calibration mode. Follow the Serial Monitor instructions to calibrate the sensor grid.

3 Flash OpenChess

Connect your ESP32 via USB and click the button below.

Your browser does not support Web Serial.
Please use Chrome, Edge, or Opera.

Web Serial access was denied.
Please grant permission and try again.
After flashing: The board will create a WiFi access point called OpenChess (password: chess123). Connect to it and open http://192.168.4.1 to configure your WiFi network and start playing.

Future updates can be installed automatically over WiFi (OTA) from the board's web interface.
